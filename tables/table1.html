Pseudo classes for links
Pseudo Class Description Example
:link Selects unvisited links. a:link { color: blue; }
:visited Selects links that have already been visited a:visited { color: purple; }
:hover Selects links when the mouse pointer hovers over them a:hover { text-decoration: underline; }
:active Selects links at the moment they are being clicked or activated. a:active { color: red; }
